tslove-tools
============

`T's LOVE <http://tslove.net>`_ に関連するツール達

Description
-----------

T's Love 用に作られたツール群です。以下のツールがあります。

imechen
  T's LOVEのプロフィール画像を差し替えます

各ツールの説明は doc ディレクトリの内容を確認してください。

Requirement
-----------

- python3
- requests

Installation
------------

::

  pip install <path>
